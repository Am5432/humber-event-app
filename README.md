# Humber Event App

A full-stack event management application developed for students and administrators to create, manage, and participate in campus events.

---

## 🚀 Features

### Student Features
- User authentication
- Browse upcoming events
- Register for events
- View event details
- Responsive mobile interface

### Admin Features
- Create and manage events
- Monitor registrations
- Manage event information
- Dashboard analytics

---

## 🛠 Technologies Used

### Mobile App
- React Native
- Expo CLI
- JavaScript

### Backend
- Node.js
- Express.js

### Database
- Firebase / MongoDB / MySQL

### Admin Dashboard
- React
- HTML/CSS
- JavaScript
